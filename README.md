# 🎸 Guitar LED Trainer

**Интеллектуальная система для обучения игре на гитаре с визуальной LED-подсветкой на грифе**

---

## 📋 О проекте

Guitar LED Trainer — это образовательная платформа для обучения игре на гитаре. Светодиодная подсветка, встроенная в гриф, в реальном времени показывает, какие струны и на каких ладах нужно зажимать.

### Как это работает

1. **База данных** содержит произведения, разбитые на квадраты (интро, куплет, припев, аутро)
2. **Каждый квадрат** состоит из последовательности аккордов с длительностью в ударах
3. **Мобильное приложение** загружает структуру, рассчитывает тайминги и отправляет команды на LED
4. **Синхронизация с 1С** позволяет методистам добавлять произведения и отслеживать прогресс

### Основные возможности

- 💡 Визуальная подсветка аккордов на грифе
- 📱 Мобильное приложение на Flutter (Android/iOS)
- 🗄️ Локальная БД SQLite для работы офлайн
- 🔄 Синхронизация с 1С:Предприятие
- 📊 Учёт прогресса учеников и статистика ошибок
- 🎼 Гибкая структура произведений (квадраты, аккорды, аппликатуры)
- 🖥️ GUI-утилиты для управления базой данных

## 🗂️ Структура базы данных

```
Произведения → Аппликатуры → Квадраты → Аккорды
```

### Таблицы

| Таблица | Описание |
|---------|----------|
| Произведения | Песни: id, название, автор, BPM |
| Аккорды | Справочник аккордов: id, название, аппликатура |
| Квадраты | Секции: id, название, аккорд1..аккорд4 |
| Аппликатуры | Порядок квадратов: id, произведение_id, квадрат1..квадрат8 |

### Формат аппликатуры аккорда

Строка из 6 символов для 6 струн (от 6-й толстой до 1-й тонкой):

- `x` — струна не играет
- `0` — открытая струна
- `1-24` — номер зажимаемого лада

**Примеры:**
- `x02210` — Am
- `xx0231` — Dm
- `020100` — E7
- `x32010` — C
- `320003` — G
- `133111` — Fm

## 📥 Установка и запуск

```bash
git clone https://github.com/sheiker12/guitar-led-trainer.git
cd guitar-led-trainer
pip install PyQt5
python3 easy_gui.py
```

### Работа с базой через консоль

```bash
sqlite3 guitar-led-trainer.db ".schema"   # структура
sqlite3 guitar-led-trainer.db ".dump"     # все данные
sqlite3 guitar-led-trainer.db "SELECT * FROM Произведения;"
```

## 🔧 Примеры SQL-запросов

```sql
-- Добавить произведение
INSERT INTO Произведения VALUES ("p4", "Новая песня", "Автор", 120);

-- Добавить аккорд
INSERT INTO Аккорды VALUES ("fm", "Fm", "133111");

-- Получить все аккорды для песни
SELECT p.название, k.название, a.название
FROM Аппликатуры ap
JOIN Произведения p ON ap.произведение_id = p.id
JOIN Квадраты k ON k.id = ap.квадрат1_id
JOIN Аккорды a ON a.id = k.аккорд1_id;
```

## 🧪 Расчёт времени для LED

Формула: `Длительность (мс) = (60000 / BPM) × количество_ударов`

При BPM = 120:
- 1 удар = 500 мс
- Аккорд на 4 удара = 2000 мс
- Аккорд на 8 ударов = 4000 мс
