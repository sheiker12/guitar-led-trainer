# 🎸 Guitar LED Trainer

Система для обучения игре на гитаре с LED-подсветкой на грифе.

## О проекте

Проект позволяет визуально обучаться игре на гитаре. LED-подсветка показывает, какие струны и лады зажимать.

## Технологии

- Flutter (Dart) - мобильное приложение
- SQLite - локальная БД
- 1С:Предприятие - серверная часть
- Python + PyQt5 - GUI

## Структура БД

Произведения → Аппликатуры → Квадраты → Аккорды

### Таблицы

- Произведения: id, название, автор, bpm
- Аккорды: id, название, аппликатура
- Квадраты: id, название, аккорд1..аккорд4
- Аппликатуры: id, произведение_id, квадрат1..квадрат8
